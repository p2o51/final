# 脳波分類プロジェクト

## 概要

このプロジェクトは、深層学習技術を用いて、被験者が画像を見ているときの脳波データから、その画像がどのクラスに属するかを分類することを目的としています。Transformerアーキテクチャを使用して、高次元の脳波データを処理し、1854クラスの中から正しいクラスを予測します。

## データセット

- **訓練セット**: 65,728 サンプル
- **検証セット**: 16,432 サンプル
- **テストセット**: 16,432 サンプル
- **入力**: 脳波データ (271 チャンネル × 281 時系列)
- **出力**: 1854 クラス

## 主な特徴

- Transformerベースのモデルアーキテクチャ
- 位置エンコーディングの使用
- ラベルスムージングを適用したクロスエントロピー損失関数
- Adam最適化アルゴリズムと重み減衰の使用



## プロジェクト構造

- `main.py`: メインのトレーニングスクリプト
- `modelv3.py`: Transformerモデルの定義
- `eval.py`: モデル評価スクリプト
- `src/`: ソースコードディレクトリ
  - `dataset.py`: データセットクラス
  - `utils.py`: ユーティリティ関数

## 結果

ベースラインモデルの検証セット正解率1.5%から、最適化後のモデルでは3.1%まで性能が向上しました。
